1. 수집 항목과 방법

직접 수집: 해당 없음. 회원가입, 서버 동기화, 클라우드 업로드 기능을 제공하지 않습니다.

자동/간접 수집:

  • 광고 식별자(IDFA 등), 광고 노출/클릭, 기기 정보, 앱 버전(AdMob SDK를 통해 수집될 수 있음)

로컬 저장(서버 전송 없음):

  • 식단 기록, 체중, 사진, 영양 계산 데이터, 30일 그래프용 통계값 등은 사용자 기기 내에만 저장됩니다.

2. 이용 목적

  • 앱 기능 제공: 식단 기록·앨범·관리, 영양성분 확인, 30일 그래프
  • 광고 제공 및 최적화: 광고 성과 측정, 부정행위 방지(AdMob)
  • 사용 편의: 로컬 데이터 보존으로 빠른 조회

3. 제3자 제공 및 처리위탁

광고 제공을 위해 Google AdMob(Google LLC)를 사용합니다. 관련 정책은 아래를 참고하십시오.

  • Google 개인정보처리방침: policies.google.com/privacy
  • AdMob 정책: support.google.com/admob

법령에 근거한 경우를 제외하고, 이용자의 개인정보를 판매하거나 임의 공유하지 않습니다.

4. 보관 및 파기

  • 로컬 데이터: 앱 삭제 또는 앱 내 “데이터 초기화” 사용 시 기기에서 완전 삭제됩니다.
  • AdMob 수집 데이터: Google 정책 및 보관기간을 따릅니다.

5. 이용자 권리와 선택

  • 광고 추적 제한: iOS 설정 > 개인정보보호 > 추적에서 조정
  • 사진 접근 권한: iOS 설정 > 개인정보보호 > 사진에서 변경
  • 데이터 삭제: 앱 삭제 또는 앱 내 초기화 기능 사용

6. 보안 조치

로컬 저장 데이터는 iOS 보안 메커니즘을 따릅니다. 전송이 필요한 경우(예: 광고 SDK 통신) 전송 구간 암호화를 사용합니다.
